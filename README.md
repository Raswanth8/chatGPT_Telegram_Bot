# ChatGPT Telegram Bot

- It uses playwright and chromium to open browser and parse html.
- Requests message query from Telegram and sends the response back to Telegram.

# How to install

```python

```

pip install -r requirements.txt

```

* Now run the server

```

python server.py

```
* Now run the Bot

```

python bot.py

